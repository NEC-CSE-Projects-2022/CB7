Dataset Details :

1️⃣ Waste Management and Recycling in Indian Cities

Source: https://www.kaggle.com/datasets/krishnayadav456wrsty/waste-management-and-recycling-in-indian-cities
Dataset Name: Waste Management and Recycling in Indian Cities

Description:
This dataset contains city-level information on municipal solid waste generation, collection, and recycling practices across Indian cities. It highlights the current status of waste management infrastructure and recycling efficiency.

Key Attributes

1.City name:
2.tal waste generated (TPD)
3.Waste collected
4.Waste recycled
5.Waste disposed
6.Recycling rate

Usage in Project:
Used to analyze waste generation patterns and recycling performance across cities and to support prediction and classification models under SDG 12.

2️⃣ Waste Management and Recycling India (Cleaned Dataset)

Source: https://www.kaggle.com/datasets/xixama/waste-management-and-recycling-india-cleaned
Dataset Name: Waste Management and Recycling India (Cleaned)

Description:
This is a preprocessed and cleaned version of waste management data for Indian cities. Missing values, inconsistencies, and formatting issues have been resolved to make the dataset suitable for machine learning applications.

Key Attributes:

1.City / State
2.Waste generated
3.Waste recycled
4.Recycling efficiency
5.Processed waste metrics

Usage in Project:
Used directly for feature engineering, clustering, and classification due to its structured and noise-free format.

3️⃣ India Area and Population Density – Census 2011

Source: https://www.kaggle.com/datasets/chaudharisanika/india-area-and-population-density-census-2011
Dataset Name: India Area and Population Density Census 2011

Description:
This dataset provides official census-based statistics on the area, population, and population density of Indian cities and states as per Census 2011.

Key Attributes:

1.City / State name
2.Total area (sq. km)
3.Population
4.Population density

Usage in Project:
Used to normalize waste generation values and derive per-capita waste indicators for better comparative analysis.

4️⃣ Indian Cities Dataset

Source: https://www.kaggle.com/code/shubhendra7/indian-cities-dataset
Dataset Name: Indian Cities Dataset

Description:
This dataset contains general metadata about Indian cities, including geographical and administrative information. It acts as a reference dataset for city mapping and data integration.

Key Attributes:

1.City name
2.State
3.Latitude and longitude
4.Tier / classification

Usage in Project:
Used for data integration, geospatial mapping, and visualization in dashboards.
